# Informations concernant les fichiers csv

## Format des fichiers CSV

Chaque ligne des fichiers CSV est formatée ainsi :
```identifiant;nom```

## Liste des fichiers CSV
- [bdd_animaux.csv](../data/bdd_animaux.csv) : Base de données contenant une liste de noms d'animaux
- [bdd_arbres.csv](../data/bdd_arbres.csv) : Base de données contenant une liste d'espèces d'arbres
- [bdd_toad.csv](../data/bdd_toad.csv) : Base de données contenant une liste de noms de champignons
- [bdd_risques.csv](../data/bdd_risques.csv) : Base de données contenant une liste de risques pouvant être présents dans une forêt
- [eau.csv](../data/eau.csv) : Base de données contenant une liste des principales rivières et lacs présents en Vendée

## Encodage

Tous les fichiers CSV doivent impérativement être ouverts et enregistrés avec l'encodage ```ISO 8859-3``` afin de garantir l'intégrité des caractères qui y sont présents
